# NUSTSec PhishGuard🛡️

> **An open-source, localized phishing simulation platform designed specifically for the African threat landscape.**

*Project in development for the Africa CyberFest 2026 - The Engine Solutions Hackathon.*

## Overview
NUSTSec PhishGuard is an open-source, localized phishing simulation platform designed specifically for the African context. While existing tools focus on Western threat vectors, PhishGuard provides pre-configured templates for localized threats such as mobile money scams, utility billing fraud, and malicious telecom promotions.
